# 字符串字面值
字符串字面值是指在C语言中用*双引号括起来的字符串*。它们是在程序编译时创建的，存储在*只读内存中*，因此被视为常量。字符串字面值是不可变的，这意味着一旦创建，就不能直接修改它们的内容。例如：

```c
const char *str = "Hello, World!";
```
